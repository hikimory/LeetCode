## Логика

1. Идея в том, чтобы добавлять ')' только после valid '('

2. Мы используем две целочисленные переменные left & right, чтобы увидеть, сколько '(' & ')' в текущей строке

3. Если left < n, то мы можем добавить '(' к текущей строке

4. Если right < left, то мы можем добавить ')' к текущей строке


```sh
				   	(0, 0, '')
				 	    |	
					(1, 0, '(')  
				   /           \
			(2, 0, '((')      (1, 1, '()')
			   /                 \
		(2, 1, '(()')           (2, 1, '()(')
		   /                       \
	(2, 2, '(())')                (2, 2, '()()')
	      |	                             |
	res.Add('(())')               res.Add('()()')
```
