## Идея

Основная идея для достижения времени $O(n)$ без деления — использовать концепцию префиксов (произведений всех элементов слева) и суффиксов (произведений всех элементов справа).

Идея в том, что для каждого элемента i итоговый результат — это (все что слева) * (все что справа).

Подробный разбор работы вашего решения на массиве nums = [1, 2, 3, 4]

Изначально имеем:

* numsLength = 4

* result = [0, 0, 0, 0]

### Шаг 1: Первый проход (слева направо) — Вычисление префиксов

Инициализация: prefixProduct = 1. Цикл идет от i = 0 до 3.

1. i = 0:
   
   * result[0] = prefixProduct $\rightarrow$ result[0] = 1
   
   * prefixProduct *= nums[0] $\rightarrow$ 1 * 1 = 1
   
   * Массив result: [1, 0, 0, 0]
  
2. i = 1:
   
   * result[1] = prefixProduct $\rightarrow$ result[1] = 1
   
   * prefixProduct *= nums[1] $\rightarrow$ 1 * 2 = 2
   
   * Массив result: [1, 1, 0, 0]

3. i = 2:
   
   * result[2] = prefixProduct $\rightarrow$ result[2] = 2
   
   * prefixProduct *= nums[2] $\rightarrow$ 2 * 3 = 6
   
   * Массив result: [1, 1, 2, 0]

4. i = 3:
   
   * result[3] = prefixProduct $\rightarrow$ result[3] = 6
   
   * prefixProduct *= nums[3] $\rightarrow$ 6 * 4 = 24 (это значение дальше не пригодится)
   
   * Массив result после 1-го цикла: [1, 1, 2, 6]
  
Как видите, теперь каждая ячейка result[i] содержит произведение всех чисел, расположенных слева от i.
### Шаг 2: Второй проход (справа налево) — Добавление суффиксов

Инициализация: suffixProduct = 1. Цикл идет в обратную сторону от i = 3 до 0.

1. i = 3:
   
   * result[3] *= suffixProduct $\rightarrow$ 6 * 1 = 6
   
   * suffixProduct *= nums[3] $\rightarrow$ 1 * 4 = 4
  
   * Массив result: [1, 1, 2, 6]

2. i = 2:
   
   * result[2] *= suffixProduct $\rightarrow$ 2 * 4 = 8
   
   * suffixProduct *= nums[2] $\rightarrow$ 4 * 3 = 12
   
   * Массив result: [1, 1, 8, 6]

3. i = 1:
   
   * result[1] *= suffixProduct $\rightarrow$ 1 * 12 = 12
   
   * suffixProduct *= nums[1] $\rightarrow$ 12 * 2 = 24
   
   * Массив result: [1, 12, 8, 6]

4. i = 0:
   
   * result[0] *= suffixProduct $\rightarrow$ 1 * 24 = 24
   
   * suffixProduct *= nums[0] $\rightarrow$ 24 * 1 = 24 (конец цикла)
   
   * Итоговый массив result: [24, 12, 8, 6]

## Итог

Программа возвращает массив [24, 12, 8, 6]. Решение абсолютно верное, оптимальное и полностью соответствует всем ограничениям задачи (время $O(n)$, память $O(1)$, без деления).
